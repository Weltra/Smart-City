> 使用AntV + Element-Plus + MapBox-GL + Echarts搭建的大屏系统

## 基本功能

系统实现了如下基本功能:

- [x] 地球自转
- [x] 控制中心
- [x] 地图复位
- [x] 事故查询
- [x] 显示事件热力图
- [x] 加载3D模型

## 🚀 项目使用
### 拉取项目
```sh
git clone https://github.com/Weltra/Smart-City.git
```
### Node.js包安装
#### 前端部分
在命令行输入以下代码，安装需要的包:
```sh
npm install 
```
#### 后端部分
后端部分在[express](./express)文件夹中，进入后端部分:
```sh
cd express
```
在命令行输入以下代码，安装需要的Node.js包:
```sh
npm install 
```
### 项目启动
#### 前端部分
使用vite来实现工程化，在命令行输入以下代码，启动项目:
```sh
npm run dev
```
#### 后端部分
进入后端部分:
```sh
cd express
```
启动后端express服务:
```sh
node index.js
```
